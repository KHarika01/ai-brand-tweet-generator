BrandVoice AI Tweet Studio:

BrandVoice AI Tweet Studio is an AI-powered tool that generates brand-consistent tweets based on a company’s tone, personality, and marketing objective.
The system analyzes brand voice and creates engaging tweets along with relevant trending hashtags.

This tool helps brands quickly generate social media content aligned with their communication style.

Live Demo :

Try the tool here:

https://ai-brand-tweet-generator-edrydnzc7rca8xrmttfthw.streamlit.app/

Features:

Brand voice analysis using AI

Generate 10 brand-aligned tweets

Campaign objective-based content (Engagement / Promotion / Awareness)

AI-generated trending hashtags

Clean and interactive UI

Copy tweet functionality

Trending tweet badge

Brand avatar/logo display

System Workflow:
The system works in the following steps:

User enters brand details

AI analyzes brand voice

AI generates tweets aligned with the brand tone

Trending hashtags are generated based on tweet content

Installation (Local Setup):

Clone the repository:

git clone https://github.com/yourusername/ai-brand-tweet-generator.git

Navigate to the project folder:

cd ai-brand-tweet-generator

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
API Configuration :

Add your Groq API key in Streamlit secrets.

Create:

.streamlit/secrets.toml

Add:

GROQ_API_KEY = "your_api_key_here"
